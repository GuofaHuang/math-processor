The examples contained in this folder can be used by using the 'Run' menu of Math Processor. Click "Run - Load File" and select one of the example files.

Brief description of some of the files is as under:

ExportTruthTable.txt: This file declares a user function that can be used to export truth tables as .html files.

truthExample.txt: An example that uses the function exportTable declared in ExportTruthTable.txt.

So load the file ExportTruthTable.txt before loading truthExample.txt.

tables.txt: This file declares a user function that can be used to create multiplication tables.

plot.txt: An interesting example that creates a Circle (Ellipse if you change window size).

StandarDeviation.txt: Shows how standard deviation can be calcuatled.

prime_test.txt: This script was used to check correctness of inprime function (The real purpose was to demonstrate some of the scripting featrues of MP!).
